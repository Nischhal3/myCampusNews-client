## BUILD

1. Clone the project

```
    git clone https://github.com/Nischhal3/myCampusNews-client.git
```

2. Install Expo Go globally

```
    npm install -g expo-cli
```

3. Install NodeJS

4. Install dependencies

```
    npm i
```

5. Run the application

```
expo start
```
